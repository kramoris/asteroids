# Asteroids Game

Simple Asteroids-style game made during Boot.dev.

## Requirements
- Python 3.13
- pygame 2.6.1

## Install
If Python 3.13 is installed:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install pygame==2.6.1
```

If not:
```bash
install uv and run:
uv venv --python 3.13
source .venv/bin/activate
uv pip install pygame==2.6.1
```

## Run
```bash
python3 main.py
```

## Controls
W / ↑ – move  
A / ← – rotate left  
D / → – rotate right  
Space – shoot
