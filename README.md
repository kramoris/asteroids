# Asteroids Game

Asteroids-style arcade game built with Python and pygame.

Built with a modular structure and gradually expanded gameplay systems.

## Requirements

* Python 3.13
* pygame 2.6.1

## Install

If Python 3.13 is installed:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install pygame==2.6.1
```

If not:

```bash
uv venv --python 3.13
source .venv/bin/activate
uv pip install pygame==2.6.1
```

## Run

```bash
python3 main.py
```

## Controls

W / ↑ – move  
A / ← – rotate left  
D / → – rotate right  
Space / Enter – shoot
